## 重置默认样式

重置一些标签的默认样式

```less
html, body, ul, li, ol, dl, dd, dt, p, h1, h2, h3, h4, h5, h6, form, fieldset, legend, img {
  margin: 0;
  padding: 0;
}
*{
  outline: 0;
  padding: 0;
  margin: 0;
}
a{
  text-decoration: none;
}
div{
  box-sizing: border-box;
}
img, input, textarea {
  border: none;
  padding: 0;
  margin: 0;
  outline-style: none;
}

ul, ol, li {
  list-style: none;
}

img {
  border: 0;
  display:block; /**/
}

i {
  font-style: normal;
}
```
